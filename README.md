##主要特性
#### ·为移动而生
URP教务系统已经落后于这个时代，移动才是未来。
法大绩点通过js/css技术保证移动端的快速和便捷。

#### ·轻量级
服务端只需要处理php的curl，所有计算在客户端由js实时完成，最大程度上减少服务器负担。使用CDN，你将能够仅仅使用极少的服务器流量和性能消耗为同学们带来最方便的服务。

#### ·可拓展性
数据的计算已经DOM转换成数组，能够极为方便地进行运算和增减功能。

#### ·快！快！快！
运用360部署的jQuery CDN，最大程度地保证程序加载速度！
